## Requirements:

Make sure you have Python 3 installed


Git Devops extension relies on `gh` - [GitHub CLI](https://cli.github.com/) tool

Install `gh` on Mac:

```brew install gh```

[Login](https://cli.github.com/manual/gh_auth_login) with your GitHub account:

```gh auth login```
